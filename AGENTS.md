# GOAL

the Goal is to have a MCP which give the full power to agents to work with computer, even if I ask for working on a 3d software, browser task and anything human can do using mouse and keyboard, I want it also to have human like behaviour like moving the mouse from its current position and not by spawning it at target position.


## backward compatibility
- we do not support any legacy code, when you change something you must not have legacy or old or anything like that.
no fucking or or nullish operator so if it didnt exist check the old one, we do NOT do that.

## code design
- Always use arrow function and modern syntax.
- each feature must be in separate folder and files must be short as possible, follow the rule of 1 file 1 function. 

## supported OS
- Windows.
- Ubuntu (not added but code base must be ready for adding this feature later).
- macOS (not added but code base must be ready for adding this feature later).

## tasks 
- only when the given task is big and long  use the `agent-task-manager-mcp` to manage the task so you wont get forget what was done and whats next or where you left off. 

## testing and scripts
- if you needed to make script for testing, you can store them in `scripts` folder.
